# Your Character Sheet

A web-based self-reflection tool styled as a life character sheet. It combines the structural wisdom of tabletop RPG character sheets with the rigour of psychological science.

## How to Use

Open `index.html` in any modern browser. No server, no build step, no dependencies required.

## Features

- Eight core life attributes scored on a 1-20 scale
- Big Five personality assessment (BFI-10)
- Custom abilities tracking
- AI prompt generator for deeper self-reflection
- Export/import your data as JSON
- Print-friendly layout
